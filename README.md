# Sales and Profit Performance Dashboard (Excel)

## Problem Statement
Businesses often struggle to identify which products, customers, and time periods drive sales and profit. Raw transactional data makes it difficult to quickly extract meaningful insights.

This project focuses on analyzing sales data and building an interactive Excel dashboard to support data-driven business decisions.

## Objective
- Analyze overall sales and profit performance
- Identify top and low performing product categories
- Understand monthly sales trends
- Compare profit contribution across years
- Enable interactive analysis using slicers

## Dataset
- Sales transaction dataset
- Contains order date, category, sales, profit, customer, and location details
- Data cleaning and structuring performed within Excel

## Tools & Techniques
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard design & formatting

## Dashboard Features
- KPI cards for Total Sales and Total Profit
- Category-wise sales analysis
- Year-wise profit comparison
- Top 5 customers by profit
- Monthly sales trend analysis
- Interactive slicers for filtering

## File Structure
- `Sales_and_Profit_Dashboard.xlsx` – Data, pivots, and dashboard in one workbook
- `screenshots/` – Dashboard image preview

## Key Insights
- A small number of categories contribute the majority of sales
- Profit trends vary significantly across years
- Sales show noticeable seasonality across months
- High-value customers contribute a disproportionate share of profit
